# Weather #

### Описание:
   Сайт, показывающий текущую погоду. 
   Информация о погоде берется с сайта https://openweathermap.org/
    

### Инструкция по запуску:

1. Перейти в директорию: 

    >frontend

2. Выполнить команду:

    >npm install

3. Выполнить команду: 

    >ng serve
